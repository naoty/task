# `dep` コマンド仕様

## 概要

タスク間の依存関係を管理するコマンド群のエントリーポイント。

## 書式

```
task dep <subcommand>
```

## サブコマンド一覧

| サブコマンド | 説明               |
| ------------ | ------------------ |
| `add`        | 依存関係を追加する |
| `delete`     | 依存関係を削除する |

## 動作

サブコマンドなしで実行した場合、利用可能なサブコマンドの一覧をエラーとして返す。

## 出力

出力形式は共通仕様（`001-cli.md`）に従う。

### サブコマンドなしで実行した場合

```json
{
  "ok": false,
  "error": {
    "message": "subcommand is required",
    "usage": "task dep <subcommand>",
    "retriable": false,
    "subcommands": [
      { "name": "add", "description": "依存関係を追加する" },
      { "name": "delete", "description": "依存関係を削除する" }
    ]
  }
}
```

## 使用例

```
$ task dep
{
  "ok": false,
  "error": {
    "message": "subcommand is required",
    "usage": "task dep <subcommand>",
    "retriable": false,
    "subcommands": [
      { "name": "add", "description": "依存関係を追加する" },
      { "name": "delete", "description": "依存関係を削除する" }
    ]
  }
}
```
