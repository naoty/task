# 詳細パネル仕様

## 概要

グラフビューでノードをクリックしたときに右からスライドインして表示されるサイドパネル。タスクの詳細閲覧・編集をおこなう。

## 表示

- ノードをクリックすると画面右端からスライドインして表示される。
- 画面全体の高さを占める。
- 初期幅は480px。
- パネルの左端にリサイズハンドルが表示され、ドラッグしてパネル幅を変更できる。
  - 最小幅: 280px
  - 最大幅: ビューポート幅の80%

## 構成

```
┌─────────────────────────────────┐
│ タイトル                      × │  ← ヘッダー
│ DOING ▾                         │
│ [tag: value] [tag: value]        │
├─────────────────────────────────┤
│                                  │
│ （リッチエディタ）               │  ← 本文エリア
│                                  │
├─────────────────────────────────┤
│ #81                              │  ← フッター
└─────────────────────────────────┘
```

## ヘッダー

### タイトル

- タスクのタイトルをテキストで表示する。
- クリックするとテキストボックスに切り替わり編集できる。
  - テキストボックスは全選択状態で開く。
  - Enter キーまたはフォーカスアウトで保存する（変更がない場合はAPIリクエストを送信しない）。
  - Escape キーで編集をキャンセルし、元のタイトルに戻す。

### ステータスセレクト

- 現在のステータスをボタンとして表示する（ステータスに対応した色）。
- クリックするとドロップダウンが開き、`TODO` / `DOING` / `DONE` から選択できる。
- 選択するとステータスが更新され、ドロップダウンが閉じる。
- パネル外をクリックするとドロップダウンが閉じる。

### 追加フィールド

- `id`, `path`, `title`, `status`, `body` 以外のフィールドが存在する場合、タグ（ピル）形式で表示する。
- 各タグにはフィールド名（大文字）と値を表示する。

### 閉じるボタン

- `×` ボタンをクリックするとパネルを閉じる。

## 本文エリア

- タスクの本文（Markdown）をリッチエディタで表示・編集できる。
- 内容が変更されると自動保存する。

## フッター

- タスクIDを `#ID` 形式で表示する。

## 閉じる方法

詳細パネルは以下の操作で閉じる。

- `×` ボタンのクリック
- Escape キーの押下
- キャンバスのクリック
