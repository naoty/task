# インデックスファイル仕様

## 概要

タスクの優先順位と依存関係をJSONファイルとして保存する。

## 保存場所

`~/.tasks/index.json`

## フォーマット

以下のフィールドを持つJSONオブジェクト。

| フィールド     | 型     | 説明                                                     |
| -------------- | ------ | -------------------------------------------------------- |
| `order`        | array  | タスクIDの配列。配列の先頭が最高優先度、末尾が最低優先度 |
| `dependencies` | object | タスクIDをキー、依存するタスクIDの配列を値とするオブジェクト |

```json
{
  "order": [3, 1, 2],
  "dependencies": {
    "1": [2, 3]
  }
}
```

上記の例では、タスク1はタスク2とタスク3に依存している（タスク2と3が完了するまでタスク1を開始できない）。

## デフォルト値

ファイルが存在しない場合は以下の値として扱う。

```json
{
  "order": [],
  "dependencies": {}
}
```
