# タスクファイル仕様

## 概要

タスクはfrontmatter付きのMarkdownファイルとして保存する。

## 保存場所

`~/.tasks/` ディレクトリ以下に保存する。

## ファイル名

`<ID>.md` の形式とする。IDは1から始まる連番の整数。

例: `1.md`, `2.md`, `3.md`

## frontmatter

以下のフィールドを持つ。

| フィールド | 型     | 説明                                      |
| ---------- | ------ | ----------------------------------------- |
| `title`    | string | タスクのタイトル                          |
| `status`   | string | タスクの状態（`todo` / `doing` / `done`） |

### 初期値

| フィールド | 初期値 |
| ---------- | ------ |
| `status`   | `todo` |

## ファイル例

```markdown
---
title: 買い物をする
status: todo
---
```
