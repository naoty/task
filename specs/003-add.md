# `add` コマンド仕様

## 概要

新しいタスクを作成するコマンド。

## 書式

```
task add <title>
```

## 引数

| 引数      | 必須 | 説明               |
| --------- | ---- | ------------------ |
| `<title>` | 必須 | タスクのタイトル   |

## 動作

1. `~/.tasks/` ディレクトリが存在しない場合は作成する。
2. 既存タスクのIDの最大値に1を加えた値を新しいIDとする。タスクが存在しない場合はIDを `1` とする。
3. タスクファイル仕様（`002-task-file.md`）に従い、`~/.tasks/<ID>.md` を作成する。
4. 作成したタスクのIDを標準出力に出力する。

## 出力

作成したタスクのIDのみを出力する。

```
<ID>
```

## 使用例

```
$ task add "買い物をする"
1
```
