# グラフビュー仕様

## 概要

タスクと依存関係をグラフとして表示するメイン画面。

## 表示

- タスクをノードとして表示する。
- 依存関係をエッジ（破線矢印）として表示する。依存先から依存元に向かって矢印が伸びる。
- ドット格子の背景を表示する。
- タスクの読み込みは `/api/graph` から取得する。

## ノード

ノードは2種類ある。

### タスクノード（子タスクを持たないタスク）

- タイトル（長い場合は末尾を切り詰めて1行で表示）
- ステータス（大文字で表示）
- タスクID（`#ID` 形式、右下に小さく表示）
- ボーダーカラーがステータスを反映する

### グループノード（子タスクを持つタスク）

- タイトル（長い場合は末尾を切り詰めて1行で表示）
- ステータス（大文字で表示）
- タスクID（`#ID` 形式、右下に小さく表示）
- ボーダーカラーがステータスを反映する
- 子タスクのノードを内包する

## ステータスの色

| ステータス | 色       |
| ---------- | -------- |
| todo       | グレー   |
| doing      | オレンジ |
| done       | グリーン |

## ノードのレイアウト

### トップレベルのノード

- 依存関係に従って左から右へ並べる（依存元が左、依存先が右）。
- 同じ列に複数のノードがある場合は、タスク一覧の順番（rootOrder）に従って縦に並べる。

### グループノード内の子ノード

- 子ノード間に依存関係がある場合は、依存順に左から右へカラム配置する。同じカラム内のノードは縦に並べる。
- 子ノード間に依存関係がない場合は、縦に並べる。

## インタラクション

- ノードはドラッグ移動できない。
- エッジの追加・削除はできない。
- ノードをクリックすると詳細パネルが開く。
- キャンバスをクリックすると詳細パネルが閉じる。

## リアルタイム更新

- `/api/events` のSSEを購読し、イベントを受け取ったタイミングでグラフを再取得して再描画する。
- 詳細パネルが開いている場合は、タスク詳細も再取得して更新する。
