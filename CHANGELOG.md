## [0.2.0] - 2026-04-02

### Features

- タスクの詳細をWeb UIのハーフモーダルで確認できるようになりました
- frontmatterのカスタムフィールドをバッジ形式で詳細UIに表示するようになりました
- 詳細UIがSSEでリアルタイムに更新されるようになりました
- 詳細パネルを開いたまま別のタスクノードをクリックすると表示が切り替わるようになりました
- 日本語フォントがOS（Windows / Mac / Linux）ごとに最適なフォントで表示されるようになりました

### Bug Fixes

- GroupNodeのタイトルが長い場合にはみ出るバグを修正しました
- GroupNodeのステータスが中央揃えにならない問題を修正しました

## [0.1.0] - 2026-03-29

### Features

- **タスク管理CLI**: `add`, `list`, `delete`, `update`, `next`, `archive`, `move` コマンドによる基本的なタスク操作
- **依存関係管理**: `dep add` / `dep delete` でタスク間の依存関係を定義・管理
- **親子タスク関係**: `--parent` オプションで階層構造を構築。子タスクのステータス変更が親タスクに自動反映
- **Web UI** (`serve` コマンド): ブラウザでタスクをDAGグラフとして視覚化
- **`--body` オプション**: `add` コマンドでタスク本文を指定可能

### Improvements

- DAGグラフのノードを依存順・優先順位順にレイアウト
- SPA資産をバイナリに埋め込み、`serve` コマンド単体で配信可能
- ポートが使用中の場合、次のポートで自動起動
